# Runibex Vestel Clone

### Kullanıllan Kütüphaneler
- Reactjs
- Tanstack Query
- React Router
- Bootstrap

### Başlangıç
Projeyi başlatmak için terminalinize
```
npm install
```
veya
```
pnpm install
```
yazdıktan sonra
```
npm run dev
```
veya

```
pnpm dev
```
yazmanız gerekmektedir
